# AIロボットは人間の夢を見るか？
## — Do AI Robots Dream of Human Dreams? —

### コンセプト

体験者はVision Proを装着し、小さなAIロボット（JetBot）の「意識」に入り込む。
ロボットの目（360°カメラ）を通して世界を見るが、その世界はAI（StreamDiffusion）によって
リアルタイムに「夢」のように変容している。

**問い：** AIが世界を「見る」とき、それは人間と同じ現実なのか？
それとも、まったく別の「夢」を見ているのか？

体験者は——AIの知覚を追体験する。

---

### 体験の三層構造

#### 第1層：現実（Raw）
JetBotのカメラが捉えるそのままの世界。
→ 体験の最初と最後に数秒だけ見せる。「これが現実です」

#### 第2層：夢（Dream）
StreamDiffusionがリアルタイムに変換した世界。
ジブリ風、サイバーパンク、水彩画、浮世絵——
AIの「知覚フィルター」を通した現実。
→ 体験のメインパート。声でスタイルを切り替えられる。

#### 第3層：覚醒（Awakening）
体験終了時、夢から現実に徐々に戻る。
スタイル強度が段階的に下がり、最後に素の映像に戻る。
Vision Proを外した瞬間——自分の目で見る世界が、少し違って見える。

---

### 体験フロー（5分）

```
0:00  暗転。パルの声が聞こえる
      「ねえ、ボクの目で世界を見てみない？」

0:15  360°映像が静かにフェードイン（Raw映像）
      目の前にJetBotの見ている世界が広がる

0:30  映像がゆっくり「夢」に変わり始める
      現実の輪郭が溶け、色が変わり、世界が再構成される
      「これがボクに見えてる世界」

1:00  ハンドジェスチャー操縦が有効に
      「手で導いて。一緒に探検しよう」
      体験者がJetBotを操縦しながら、夢の世界を探索

2:00  声でスタイル変更可能に
      「ジブリ」「サイバーパンク」「浮世絵」……
      AIの「夢」が変わる
      「ほら、同じ場所なのに全然違う世界に見えるでしょ？」

4:00  映像が徐々にRawに戻り始める
      「そろそろ目を覚ます時間だよ」
      夢が溶けて、現実が戻ってくる

4:30  完全にRaw映像。JetBot停止。
      「……ねえ、キミの目で見る世界は、どんな色？」

5:00  暗転。体験終了
```

### 演出要素

#### パルのナレーション（TTS）
- ElevenLabs Yuiボイスで、体験の要所でパルが語りかける
- 押し付けがましくなく、詩的に
- 日本語 + 英語字幕（外国人来場者対応）

#### 音響
- JetBotの環境音（マイク）+ アンビエントBGM
- スタイル変更時に対応する音楽が変わる
  - ジブリ → アコースティック
  - サイバーパンク → エレクトロニック
  - 浮世絵 → 和楽器

#### 走行エリア
- 3m × 3m のミニチュア世界
- 小さな建物、植物、鏡、光源を配置
- 「ロボットにとっての大冒険」が人間にとっては小さなテーブルの上

#### 外部モニター（非体験者向け）
- 左: JetBotの走行（実物、上から見下ろせる）
- 中: 体験者が今見ている360°映像（夢の世界）
- 右: Raw映像（現実）
- 観客は「現実」と「AIの夢」を同時に比較できる

---

### テーマの深度

#### 表層：エンターテインメント
「すごい！綺麗！面白い！」— 純粋な視覚体験として楽しめる

#### 中層：テクノロジー体験
「AIってこうやって画像を変換するんだ」— 技術デモとして

#### 深層：哲学的問い
「AIの知覚と人間の知覚の違いは何か？」
「私たちも、脳というフィルターを通して"夢"を見ているのでは？」
「現実とは何か？」

### 技術スタック

| 要素 | 技術 | 役割 |
|------|------|------|
| ロボット | JetBot + Jetson Nano | 走行体 |
| 360°カメラ | RICOH THETA S | 環境映像取得 |
| 障害物検知 | CSI IMX219 + ResNet18 | 安全走行 |
| AI変換 | StreamDiffusion (RTX 2080Ti) | リアルタイムスタイル変換 |
| 没入体示 | Apple Vision Pro | 360°没入表示 |
| 操縦 | ハンドジェスチャー + MQTT | 体験者の操作 |
| ナレーション | ElevenLabs TTS (パル) | 語りかけ |
| 音響 | アンビエントBGM + 環境音 | 世界観演出 |
| 通信 | 専用Wi-Fiルーター | 安定接続 |
