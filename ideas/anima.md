# Anima — アニミズムARカメラ
*すべてのものに魂を宿す*

## コンセプト
カメラで物体をスキャン → 3D変換 → 特徴と文脈を理解 → 自動リギング → 声を生成 → 物と会話できる

## パイプライン
1. **カメラスキャン**: ARKit / 単眼カメラ
2. **物体認識**: VLM（GPT-4o / Gemini）で物体の種類・素材・状態・用途・文脈を分析
3. **3D再構築**: SHARP（単眼→3DGS, 4秒）/ Apple Object Capture / NeRF
4. **性格生成**: 物の特徴→LLMで性格プロンプト生成（古い→渋い、新しい→元気、壊れかけ→弱々しい）
5. **自動リギング**: 3Dモデルに顔パーツ配置（口、目）→ BlendShape → 表情アニメーション
6. **声生成**: ElevenLabs Voice Design API で性格に合った声を自動生成
7. **AR表示**: 物の上に3Dキャラ表示 + リップシンク + 会話

## 会話例
- 古い椅子: 「腰痛いよ〜もう30年座られてるからね」（渋い声）
- マグカップ: 「今日のコーヒー熱すぎない？」（明るい声）
- 観葉植物: 「もうちょっと日当たり欲しいな…」（穏やかな声）
- 鍵: 「また忘れられるとこだった！」（焦った声）
- 古本: 「最後に開かれたの3年前だよ…」（寂しげな声）

## 技術スタック
- ARKit + RealityKit（iOS / visionOS）
- SHARP（3DGS生成）← VisionPALと共有
- VLM（物体認識+性格生成）
- ElevenLabs Voice Design（声自動生成）
- LLM（会話エンジン）
- BlendShape / リギング自動化

## VisionPALとの共通点
- SHARP 3DGS パイプライン
- ARKit ヘッドトラッキング
- ElevenLabs TTS
- MetalSplatter レンダリング
- Swift / RealityKit

## 差別化
- 既存AR: 物を**認識**して情報表示（Google Lens等）
- Anima: 物に**人格**を与えて**会話**する
- アニミズム（万物に魂）× テクノロジー = 新しいAR体験

## 展開案
- 博物館: 展示品が自分の歴史を語る
- 教育: 理科の実験器具が使い方を教える
- アート: インスタレーション — 部屋の全ての物が喋り出す
- 子供向け: ぬいぐるみや玩具との会話
- 高齢者: 遺品・思い出の品との対話
