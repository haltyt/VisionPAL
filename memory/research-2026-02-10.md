# 🔬 夜間リサーチ 2026-02-10（テーマ: AI × 音楽 / クリエイティブ）

## 見つけたもの

### 1. 🎵 Obsidian Neural (ai-dj) — リアルタイムAI音楽生成VSTプラグイン
- **GitHub:** https://github.com/innermost47/ai-dj
- テキストプロンプトからリアルタイムでループ生成するVST3プラグイン
- 8トラックサンプラー、MIDIトリガー(C3-B3)、16ステップシーケンサー
- 「Draw-to-Audio」機能: スケッチを描くとAIが音楽に変換
- DAWのテンポに完全同期、ライブパフォーマンス向け
- AIMLA 2025で発表（Queen Mary University London）
- ⭐ Sunoみたいな「曲生成」ではなく「演奏ツール」としてのAI

### 2. 🎶 HeartMuse — ローカルAI音楽ジェネレーター
- **GitHub:** https://github.com/strnad/HeartMuse
- HeartMuLa（OSSの音楽生成モデル）+ Ollama/OpenAIでスマート歌詞生成
- 完全ローカル実行、Gradio Web UI
- スタイルトランスファー（リファレンスオーディオから影響）
- 最大240秒の楽曲生成、バッチ生成対応
- ⭐ ローカルで動くのが魅力。GPU要求がJetsonで行けるか要確認

### 3. 🔊 LocalSoundsAPI — ポータブルオフラインAIオーディオスタジオ
- **GitHub:** https://github.com/rookiemann/LocalSoundsAPI
- XTTS v2、Fish Speech、Kokoro（TTS）+ Stable Audio + ACE-Step（音楽生成）
- Whisper文字起こし、ローカルLLMチャットボット内蔵
- REST API + Web UI、完全オフライン
- モデルごとにCPU/GPU選択可能
- ⭐ TTS+音楽+文字起こしのオールインワン。パルの声周りの強化に使えるかも

### 4. 🤖 LingBot-VA — Causal World Modeling for Robot Control
- **arXiv:** https://arxiv.org/abs/2601.21998
- ビデオワールドモデル + ビジョン言語事前学習でロボット制御
- 自己回帰拡散フレームワーク: フレーム予測とポリシー実行を同時学習
- Mixture-of-Transformers (MoT) アーキテクチャ
- 閉ループロールアウト + 非同期推論パイプライン
- ⭐ 音楽じゃないけど、JetBotの自律制御に直結する論文！

### 5. 📝 ACE-Step — 高品質AI音楽生成モデル (3.5B)
- HeartMuse、LocalSoundsAPI等で使われている音楽生成エンジン
- マルチラインプロンプト（スタイル+歌詞）対応
- OSSで利用可能、複数のフロントエンドが登場中
- ⭐ 2025-2026年のOSS音楽生成のデファクトスタンダードになりつつある

## 💡 パル＆ハルトのJetson環境で実現できそうなアイデア

### アイデア1: ギターセッションv6 — AIコード検出→リアルタイム楽曲生成
現在のguitar_session.py（コード検出→ドラム+ベース伴奏）を進化させて：
- ACE-StepかHeartMuLaの軽量版をサーバー側で動かす
- Jetsonでコード進行を検出→APIで楽曲ループをリクエスト
- 生成されたループをBTスピーカーで再生（現行のドラム+ベースの代わりに）
- 「今日はジャズ風で」「ボサノバにして」等、Discordから雰囲気指定

### アイデア2: パルの「作曲モード」
- ハルトが弾いたギターフレーズをマイクで録音
- Whisperで音声→テキスト変換（ハミング/歌詞検出）
- HeartMuseで楽曲に展開
- パルが「こんな曲にしてみたよ！」とDiscordで共有
- JetBotのスピーカーからデモ再生
