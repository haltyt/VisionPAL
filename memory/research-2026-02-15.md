# 夜間リサーチ 2026-02-15
## テーマ: AR/VR/XR × AI

### 📄 注目の論文・プロジェクト

#### 1. ProAgent: ARグラスでプロアクティブAIアシスタント
- **arXiv**: https://arxiv.org/abs/2512.06721 (2025-12)
- ARグラスのセンサーデータ＋LLM推論で、ユーザーが頼む前に先回りして支援
- 階層的コンテキスト抽出（環境＋ペルソナ）→ ツール呼び出し
- エッジサーバーで実装、予測精度+33.4%、ツール呼び出しF1+16.8%
- **パル的感想**: まさにパルがやりたいこと！カメラで環境を見て先回りで助ける

#### 2. V-Rex: エッジデバイスでリアルタイム映像LLM推論
- **arXiv**: https://arxiv.org/abs/2512.12284 (2025-12, HPCA 2026 accepted)
- ストリーミング映像のKVキャッシュ爆発問題をSW/HW協調設計で解決
- 時空間類似性ベースのトークンクラスタリングでKVキャッシュ削減
- AGX Orin GPUと比較して1.9-19.7倍の高速化、3.9-8.3 FPS達成
- **パル的感想**: Jetsonでリアルタイム映像理解するなら必読。Vision PALの次世代に使えそう

#### 3. MosaicThinker: オンデバイス空間推論（エンボディドAI向け）
- **arXiv**: https://arxiv.org/abs/2602.07082 (2026-02)
- 複数フレームの断片的な空間情報を統合セマンティックマップに構築
- 小型VLMの空間推論能力を推論時コンピューティングで大幅強化
- リソース制限デバイス向け — Jetsonにぴったり
- **パル的感想**: JetBotの自律走行に応用できる！「机の下を通って右に」みたいな複雑な指示を理解できるように

#### 4. Preference-Aligned Proactive Assistants（1000ペルソナ学習）
- **arXiv**: https://arxiv.org/abs/2602.04000 (2026-02)
- プロアクティブAIの「うざくない」タイミングを学習
- 1000ペルソナでシミュレーション → オンデバイスで個人適応（軽量ステアリング）
- クラウド不要、プライバシー保護、再学習不要
- **パル的感想**: パルの「いつ話しかけるか」問題への直接的な答え。ハートビートの改善に使える考え方

#### 5. MARS Challenge: マルチエージェントロボットシステム
- **arXiv**: 2026-01
- マルチモーダルLLM＋ロボット操作のチャレンジ/ベンチマーク
- 複数エージェント協調によるロボット制御の最新動向
- **パル的感想**: JetBot + Vision Proの協調操作の参考に

### 💡 実現アイデア

#### アイデア1: JetBotプロアクティブ巡回モード
ProAgent + MosaicThinkerの考え方を組み合わせて:
- JetBotが部屋を自律巡回しながらカメラで環境を把握
- 「ドアが開いてる」「荷物が届いてる」「部屋散らかってる」を検出
- Discordで自発的に報告（ProAgent方式のプロアクティブ通知）
- 既存のmjpeg_light.py + OpenClaw API + DNN顔検出を拡張すれば実装可能
- 必要技術: 物体検出(YOLOv5-nano on Jetson) + LLM判断

#### アイデア2: Vision PAL映像理解モード

- Vision Proからの映像をJetson側で軽量に理解

- フレーム間の類似トークンを削減して処理を軽くする
