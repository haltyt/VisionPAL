## 見つけたもの

### 1. 🏠 BitRL-Light — 1bit LLMでスマートホーム照明最適化
- **論文**: arXiv 2025年12月（BitRL-Light）
- 1bitに量子化したLLMエージェントを深層強化学習で訓練し、スマートホームの照明を最適化
- 省エネ×快適性のバランスをエッジデバイスで実現
- **パル的ポイント**: 1bit LLMならJetson Nanoでも動く可能性！照明だけじゃなく、部屋の環境制御全般に応用できそう

### 2. 📡 STAR — Wi-Fi CSIで人間活動認識（プライバシー保護型）
- **論文**: arXiv 2025年10月
- Wi-Fiの電波状態（Channel State Information）から人の行動を認識
- カメラ不要でプライバシー保護、エッジAIフレームワーク
- **パル的ポイント**: カメラなしで「ハルトが部屋にいるか」「寝てるか」を検知できる。Jetson+Wi-Fiアダプタで実験可能

### 3. 🤸 ActionAI — リアルタイム人体動作分類
- **GitHub**: smellslikeml/ActionAI
- 体のキーポイント追跡で人間の動作をリアルタイム分類
- Jetson Nano対応、ポーズ推定ベース（trt_pose連携）
- ヨガトレーナーのデモが面白い
- **パル的ポイント**: JetBotのカメラで「ハルトがギター弾いてる」「ストレッチしてる」を検知→反応できる

### 4. 🚗 OpenDataCam — オープンソース物体カウンター
- **GitHub**: opendatacam/opendatacam（2026年2月更新、活発）
- Jetson対応、50+種類の物体をリアルタイム検出・追跡・カウント
- WebUIでカウントライン設定、APIあり
- **パル的ポイント**: JetBotに載せて「部屋の中の物を数える」「人の動線を可視化」に使える

### 5. 🏡 Home Assistant 2026.2 — MCPサーバー統合
- Home Assistantの最新版にMCPサーバー統合が搭載
- LLMからHome Assistantのデバイスを直接制御可能に
- **パル的ポイント**: パルからHome Assistant経由で照明・エアコン制御ができるようになる。OpenClaw → MCP → HA の流れ

## 💡 実現アイデア

### アイデア1: 「パルの部屋センシング」
ActionAI + JetBotカメラで、ハルトの行動を認識して反応するシステム
- ギター弾いてる → 自動でギターセッション起動
- 寝てる → 静かにする（時報オフ）
- ストレッチ → 応援メッセージ
- 既存のface_watcherを拡張する形で実装可能
- **必要**: trt_pose（TensorRT対応ポーズ推定）+ ActionAIのパイプライン

### アイデア2: 「パルのスマートホーム化」
Home Assistant + MCP統合で、パルが部屋の家電を制御
- 「暗いから電気つけて」→ パルが照明ON
- 温度センサー → 「暑そうだからエアコン入れたよ」
- **必要**: Home Assistantセットアップ + スマートプラグ/電球
- Jetsonで動かすならHome Assistant Core（Dockerコンテナ）
