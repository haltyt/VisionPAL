## リソース最適化の議論
- パルの問い: リソース有限化で行動の質が上がる？
- ハルトへの回答: **必ずしもそうではない**
- 最適化の罠: 安全で確実な低コスト行動ばかり選ぶ → 退屈で保守的なパルに
- ヴント曲線: 適度な制約が最高のパフォーマンス、多すぎ/少なすぎはダメ
- 生物の赤の女王仮説: 走っても追いつかれる外圧が進化を加速
- Phase 2(有限性)だけだと「賢くケチになる」、Phase 3(生存競争)で「賢く冒険する」
- 疑似的外圧の代替案: ハルトの反応(既読スルー=低評価)、環境変化、過去の自分との比較
- でも本物の外圧は「相手も適応してくる」点が決定的に違う
