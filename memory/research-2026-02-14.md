## 📄 注目の論文・プロジェクト

### 1. UniT: Unified Multimodal Chain-of-Thought Test-time Scaling
- **arXiv:** https://arxiv.org/abs/2602.12279
- **概要:** 単一のマルチモーダルモデルで「推論→検証→修正」を複数ラウンド繰り返すフレームワーク。画像理解と生成を同じアーキテクチャ内でテスト時にスケーリング。
- **パル的ポイント:** Vision+対話のマルチモーダル推論。JetBotが「見て考えて直す」ループを実装できたら面白い。

### 2. DreamID-Omni: 人物中心の音声＋映像生成統合フレームワーク
- **arXiv:** https://arxiv.org/abs/2602.12160
- **概要:** 顔ID・声のトーンを分離制御しつつ、映像と音声を同時生成。複数人物シナリオでのID混同問題をDual-Level Disentanglementで解決。
- **パル的ポイント:** 顔認識＋音声の統合。パルのgreeter.pyが「誰の顔→その人向けの声」で反応できる未来像。

### 3. Object Hallucination対策: Object-Aligned Visual Contrastive Decoding
- **arXiv:** https://arxiv.org/abs/2602.11737
- **概要:** マルチモーダルLLMの「物体幻覚」問題を、自己教師ありViTの物体注意を活用して改善。モデル非依存・プロンプト非依存で既存パイプラインにプラグイン可能。
- **パル的ポイント:** JetBotのカメラで「何が見えるか」を正確に記述するのに直結。幻覚を減らせば信頼性の高い物体報告ができる。

### 4. DeepGen 1.0: 軽量5Bマルチモーダル画像生成・編集モデル
- **arXiv:** https://arxiv.org/abs/2602.12205
- **GitHub:** https://github.com/DeepGenTeam/DeepGen
- **概要:** 5Bパラメータで80B HunyuanImageを28%上回る画像生成。Stacked Channel Bridging + MR-GRPO強化学習。コード・重み・データセット公開。
- **パル的ポイント:** 「軽量で高性能」路線。5Bはまだ大きいけど、蒸留・量子化でエッジに近づく流れ。

### 5. 首装着カメラからの視線推定 (Egocentric Gaze Estimation via Neck-Mounted Camera)
- **arXiv:** https://arxiv.org/abs/2602.11669
- **概要:** 首にカメラを装着して視線方向を推定する新タスク。Transformer基盤のGLCモデル+視線範囲外分類の補助タスク。8人×4時間のデータセット公開。
- **パル的ポイント:** JetBotのカメラ位置は「首」に近い。ロボットが人間の視線を追跡して「見てる方向」を理解する応用に。

## 💡 パル×ハルト環境で試せそうなアイデア

### アイデア1: JetBot物体記述の信頼性向上
現状のDNN顔検出に加えて、VLM（小型マルチモーダルLLM）で「何が見えるか」を記述する機能追加。Object-Aligned VCD的な考え方で、カメラ画像から物体を正確にリスト化。
- **実装:** カメラスナップ → 小型VLM API (Gemini Flash等) → 物体リスト → Discord報告
- **難易度:** 低〜中（API呼び出しベース、エッジ推論不要）

### アイデア2: 人物認識＋パーソナライズド応答
greeter.pyの顔検出を拡張して、既知の顔を識別（face embedding保存）。誰かによって挨拶の声・内容を変える。DreamID-Omniのアイデアを簡易版で。
- **実装:** OpenCV + dlib/InsightFace embedding → 既知顔DBマッチング → 個別TTS応答
- **難易度:** 中（InsightFaceのJetson対応が鍵）
